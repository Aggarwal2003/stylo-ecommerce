# Stylo E-Commerce Website

This is a full-stack e-commerce website built using the MERN stack.

The project contains:
- A frontend for users
- An admin panel
- A backend API

## Tech Used
- React
- Redux
- Node.js
- Express
- MongoDB

## Project Structure

frontend  - user website  
admin     - admin dashboard  
backend   - server & database  

## How to Run Locally

Backend:
cd backend
npm install
npm run dev

Frontend:
cd frontend
npm install
npm run dev

Admin:
cd admin
npm install
npm run dev
